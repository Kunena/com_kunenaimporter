## ABOUT

Kunena Importer lets you import data from other forums like 
* Agora
* CCBoard
* Discussions
* JoomBB
* NinjaBoard

* phpBB3
* phpBB2
* PNphpBB2
* SMF2

## REQUIREMENTS

Requirements: Joomla 1.5 / 2.5, Kunena Forum 2.0

## USAGE

Install com_kunenaimporter via the Joomla extension manager.
